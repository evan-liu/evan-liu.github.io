[Finicky](https://github.com/johnste/finicky) allows you to set up rules that decide which browser is opened for every link or url.
Set [Finicky](https://github.com/johnste/finicky) as the default browser and write the rules in `~/.finicky.js`.

[Finicky](https://github.com/johnste/finicky) makes sure links in IMs (e.g. `Slack`) and email clients opens the right browser.

```javascript title="~/.finicky.js"
module.exports = {
  defaultBrowser: 'Safari',
  handlers: [
    {
      match: finicky.matchHostnames([
        /\bgithub\.com$/, // Big PRs break Safari
        'localhost', // Chrome dev tool is better
        // ... other work and dev sites
      ]),
      browser: 'Google Chrome',
    },
  ],
}
```
