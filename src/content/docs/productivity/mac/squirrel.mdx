[Squirrel](https://github.com/rime/squirrel) (鼠鬚管, [Rime](https://rime.im/) for macOS) is a Chinese input source.

:::tip[简体输入]
按組合鍵 ``Ctrl+` `` 或 `F4` 鍵喚出輸入方案選單
:::

👨‍💻 第一次使用鼠鬚管输入法时完全被它的响应速度惊呆了，几乎就是零延迟。再也无法使用其它中文输入法了。
